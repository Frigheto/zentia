Política de Privacidade

Esta Política de Privacidade descreve como a ZENT I.A. LTDA ("Empresa", "nós") coleta, usa, armazena e protege os dados pessoais dos usuários ("Titular"), em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei 13.709/2018).

1. Dados Coletados

1.1. Dados fornecidos pelo Titular

  • Dados de cadastro: nome, e-mail, telefone/WhatsApp, nome do escritório;
  • Dados de pagamento: informações de cartão de crédito (processadas por terceiros — Stripe);
  • Dados profissionais: área de atuação, número de advogados, OAB.

1.2. Dados coletados automaticamente

  • Dados de navegação: endereço IP, tipo de navegador, páginas visitadas, tempo de permanência;
  • Cookies: cookies essenciais, de desempenho e de marketing (mediante consentimento);
  • Dados de uso: funcionalidades utilizadas, frequência de acesso, interações na Plataforma.

2. Finalidade do Tratamento

Os dados pessoais são tratados para as seguintes finalidades:

  • Execução contratual: fornecer e manter os serviços contratados;
  • Comunicação: enviar notificações, atualizações e suporte;
  • Melhoria do serviço: analisar uso para aprimorar funcionalidades;
  • Marketing: enviar comunicações comerciais (mediante consentimento);
  • Segurança: prevenir fraudes e proteger a Plataforma;
  • Obrigações legais: cumprir exigências regulatórias e fiscais.

3. Base Legal (Art. 7 da LGPD)

  • Consentimento (Art. 7, I): cookies de marketing e comunicações comerciais;
  • Execução de contrato (Art. 7, V): dados necessários para prestação do serviço;
  • Legítimo interesse (Art. 7, IX): analytics e melhoria contínua;
  • Obrigação legal (Art. 7, II): dados fiscais e regulatórios.

4. Compartilhamento de Dados

Os dados podem ser compartilhados com:

  • Processadores de pagamento: Stripe, para processamento de transações financeiras;
  • Plataforma de base: GoHighLevel, para funcionalidades de CRM, automações e comunicação;
  • Serviços de analytics: Google Analytics e Meta, para análise de comportamento (mediante consentimento);
  • Autoridades legais: quando exigido por lei ou ordem judicial.

Não vendemos, alugamos ou comercializamos dados pessoais a terceiros.

5. Armazenamento e Segurança

5.1. Os dados são armazenados em servidores seguros com criptografia em trânsito (TLS/SSL) e em repouso.

5.2. Adotamos medidas técnicas e organizacionais para proteger os dados contra acesso não autorizado, perda ou destruição.

5.3. O acesso aos dados é restrito a colaboradores autorizados que necessitam deles para o desempenho de suas funções.

6. Retenção de Dados

  • Dados de conta ativa: mantidos enquanto a conta estiver ativa;
  • Após encerramento: dados mantidos por 30 dias para recuperação, depois excluídos;
  • Dados fiscais: mantidos pelo prazo legal de 5 anos;
  • Dados de analytics: anonimizados após 26 meses.

7. Direitos do Titular (Art. 18 da LGPD)

O Titular tem direito a:

  • Confirmação e acesso: saber se seus dados são tratados e acessar uma cópia;
  • Correção: solicitar a correção de dados incompletos ou desatualizados;
  • Anonimização ou eliminação: solicitar a exclusão de dados desnecessários;
  • Portabilidade: receber seus dados em formato estruturado;
  • Revogação do consentimento: retirar consentimento a qualquer momento;
  • Oposição: opor-se ao tratamento em determinadas situações;
  • Informação sobre compartilhamento: saber com quem seus dados são compartilhados.

Para exercer seus direitos, entre em contato pelo e-mail: privacidade@zentia.com.br

8. Cookies

Utilizamos os seguintes tipos de cookies:

  • Essenciais: necessários para o funcionamento básico do site (não requerem consentimento);
  • Desempenho: coletam dados de uso anônimos para melhorar a experiência (requerem consentimento);
  • Marketing: utilizados para exibir anúncios relevantes (requerem consentimento).

Você pode gerenciar suas preferências de cookies a qualquer momento através do banner de consentimento disponível no site.

9. Transferência Internacional

Alguns dos nossos processadores (Stripe, Google, Meta) podem processar dados em servidores localizados fora do Brasil. Nesses casos, garantimos que a transferência ocorre com nível adequado de proteção, conforme exigido pela LGPD (Art. 33).

10. Encarregado de Proteção de Dados (DPO)

O Encarregado pelo tratamento de dados pessoais pode ser contatado pelo e-mail: dpo@zentia.com.br

11. Alterações nesta Política

Esta Política pode ser atualizada periodicamente. Notificaremos os Titulares sobre alterações relevantes por e-mail ou pela Plataforma. A data de última atualização será sempre indicada no topo deste documento.

12. Contato

Para dúvidas, solicitações ou reclamações sobre privacidade e proteção de dados:

  • E-mail: privacidade@zentia.com.br
  • DPO: dpo@zentia.com.br

Caso não fique satisfeito com nossa resposta, você pode apresentar reclamação à Autoridade Nacional de Proteção de Dados (ANPD).

Nota: Este documento é um modelo base conforme a LGPD (Lei 13.709/2018) e deve ser revisado por um advogado antes da publicação definitiva.
