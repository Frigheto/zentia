## User Story
**Como** visitante preocupado com privacidade,
**Eu quero** acessar os Termos de Uso e Política de Privacidade da ZENT I.A.,
**Para que** eu saiba como meus dados serão tratados antes de me cadastrar.

## Acceptance Criteria

- [ ] **AC-1:** Página `/termos` com Termos de Uso completos
- [ ] **AC-2:** Página `/privacidade` com Política de Privacidade conforme LGPD
- [ ] **AC-3:** Links no footer da landing page apontam para as páginas
- [ ] **AC-4:** Páginas seguem o mesmo design system (dark theme, tipografia)
- [ ] **AC-5:** Conteúdo cobre: coleta de dados, uso, compartilhamento, direitos do titular, cookies, contato DPO
- [ ] **AC-6:** Data de última atualização visível nas páginas
