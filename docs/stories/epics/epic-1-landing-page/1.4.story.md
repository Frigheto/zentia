## User Story
**Como** cliente assinante da ZENT I.A.,
**Eu quero** clicar em "Entrar" na landing page,
**Para que** eu seja redirecionado para a plataforma GoHighLevel onde acesso meu painel.

## Acceptance Criteria

- [ ] **AC-1:** Botão "Entrar" no header redireciona para URL do subdomínio GHL (ex: app.zentia.com.br)
- [ ] **AC-2:** URL de redirect é configurável via variável de ambiente
- [ ] **AC-3:** Link abre em nova aba (target="_blank" com rel="noopener noreferrer")
- [ ] **AC-4:** Botão "Entrar" é visível e acessível em mobile (menu hamburger)
- [ ] **AC-5:** Se o usuário já está logado no GHL, vai direto para o dashboard

## Scope

### IN
- Configuração do link de login
- UX do botão em desktop e mobile

### OUT
- Página de login customizada
- SSO ou integração de autenticação
- Gestão de sessão
