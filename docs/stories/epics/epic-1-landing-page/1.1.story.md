## User Story
**Como** visitante que busca soluções para gestão de escritório de advocacia,
**Eu quero** encontrar a ZENT I.A. nos resultados de busca do Google,
**Para que** eu possa conhecer a plataforma e avaliar se atende minhas necessidades.

## Acceptance Criteria

- [ ] **AC-1:** Meta tags de título e descrição estão presentes e otimizadas para o nicho jurídico
- [ ] **AC-2:** Open Graph tags configuradas (og:title, og:description, og:image, og:url)
- [ ] **AC-3:** Twitter Cards configuradas (twitter:card, twitter:title, twitter:description, twitter:image)
- [ ] **AC-4:** Structured data (Schema.org) implementada: Organization, Product, FAQ
- [ ] **AC-5:** Sitemap.xml gerado e acessível
- [ ] **AC-6:** robots.txt configurado corretamente
- [ ] **AC-7:** Tags semânticas HTML5 (header, main, section, footer, nav, article) estão corretas
- [ ] **AC-8:** Alt texts em todas as imagens
- [ ] **AC-9:** Heading hierarchy correta (h1 > h2 > h3 sem pulos)
- [ ] **AC-10:** URL canônica configurada

## Scope

### IN
- Meta tags (title, description, OG, Twitter)
- Structured data (JSON-LD)
- Sitemap.xml e robots.txt
- Semântica HTML
- Alt texts

### OUT
- Google Search Console setup (infraestrutura)
- Link building
- Conteúdo de blog/SEO off-page
