# Plano de Implementação - ZENT I.A. (Minimalismo Global)

**Objetivo:** Reduzir carga cognitiva. Textos curtos, diretos e layout espaçado.

## 1. Redução de Texto (Copywriting Minimalista)
*   **Hero:**
    *   *De:* "Centralize Marketing, Comercial e Gestão de Leads. A ZENT I.A. coloca ordem no caos..."
    *   *Para:* "Marketing, Comercial e Gestão. 100% Organizado."
*   **Problema:**
    *   *De:* "Ferramentas desconectadas, planilhas manuais..."
    *   *Para:* "O Caos custa caro. A Organização escala."
*   **Features:**
    *   Substituir parágrafos por bullets de 3 palavras no máximo.

## 2. Layout Estratégico & Design (CSS)
*   **Whitespace:** Aumentar `gap` entre elementos de 2rem para 4rem ou mais.
*   **Foco Visual:**
    *   Usar "Glass Cards" isolados para destacar features chave, em vez de listas densas.
    *   Alinhar textos à esquerda (mais natural para leitura rápida) em todas as seções, mantendo o padrão do Hero.

## 3. Execução
1.  Editar `index.html` para "podar" o conteúdo.
2.  Ajustar `style.css` para maximizar o respiro (`padding` nas sections).
